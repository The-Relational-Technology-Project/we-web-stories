# We Web Stories

A community storytelling platform about reimagining the web — visitors share and browse visions for a more relational, human-centered internet.

## What is this?

We Web Stories puts the "we" back in the web. It's a space where people submit short stories and visions about what a community-connected internet could look like, and browse what others have imagined. A collective dreaming space for the relational web.

## Features

- **Story browsing** — Read visions from others about a more relational web
- **Story submission** — Share your own vision for community-centered technology
- **Story grid** — Browse the growing collection of community contributions

## Tech stack

- [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/) + [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/) + [shadcn/ui](https://ui.shadcn.com/)
